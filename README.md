# IT Encyklopedie

Jednoduchá webová encyklopedie zaměřená na základní pojmy z oblasti informačních technologií.

## Obsah

- Hlavní stránka s úvodem
- Navigační menu
- Vyhledávání pojmů
- 5 samostatných stránek s vysvětlením pojmů:
  - Internet
  - CPU
  - RAM
  - GPU
  - SSD

## Použité technologie

- HTML5
- CSS3
- JavaScript



## Spuštění

1. Stáhněte nebo naklonujte repozitář.
2. Otevřete soubor `index.html` v internetovém prohlížeči.
3. Procházejte jednotlivé pojmy pomocí menu nebo vyhledávání.
